BC省雇主担保移民

BC省概况

BC省的英文全称为British Columbia，也即不列颠格哥伦比亚省，也称卑诗省。BC省位于加拿大西岸，濒临太平洋，属于温带海洋性气候，受纬度、山区地形以及太平洋影响，呈现多变的面貌。

BC省的首府是位于温哥华岛上的维多利亚，这里是一个极富英国风情的小岛，位于温哥华岛南岸，人口30多万。维多利亚气候极佳，全年平均每日有6小时日照，最冷的日子也很少低于摄氏零度。

温哥华位于BC省的西南海岸，人口57万。该市是太平洋沿岸最大的港口城市，也是国际贸易的重要中转站。温哥华背山面海，气候温和，四季如春，有花园城市之美誉。温哥华港是北美第二大港，为加拿大通往亚太地区的主要门户。以温哥华市为中心，由20多个市、区组成的大温地区（Greater Vancouver）约有人口200多万，华人约40万，是仅次于英裔族群的第二大族群。

飞出国立足于加拿大BC省，在当地有丰富的雇主资源，一手对接，在源头上把控雇主资质，同时还为登陆BC省的客户提供各种地接服务，让您在BC省即将开启的新生活毫无后顾之忧。

BC省雇主担保移民项目适合什么样的申请人

BC省是加拿大的经济大省，人口密集，支柱产业为林业、矿业、农渔业，新兴产业有旅游业、电影业和高科技产业等。BC省的高科技产业在促进本省经济发展方面发挥着重要作用，对高技术人才的需求非常大，因此就业机会也相对很多。

BC省雇主担保移民项目的申请条件相比于海洋四省及萨省等省份稍高，有本省自己的评分标准，因此并不是所有申请人都适合申请。

BC省雇主担保移民更适合雅思在5分以上，本科以上学历，职业类别属于0、A或B类，近十年有至少2年相关工作经验的申请人。

ICT类职业申请人以及一些中高级管理职位的申请人非常适合申请BC省，针对ICT类职业BC省给予了非常大的支持，开辟了BC PNP Tech这条绿色通道，让这类申请人能够以更低的分数在更短的时间内获得更多的机会。

BC省雇主担保移民项目介绍

加拿大BC省雇主担保移民是由BC省政府和联邦移民部协议共同运作的省提名（PNP）移民项目，旨在帮助BC省的雇主雇佣并保留拥有所需技能的合格外国技术工人。

BC省雇主担保移民，首先由企业筛选合格的申请人，并同意给申请人下发offer，通过省提名计划支持申请人申请永久居留；然后由雇主和申请人向PNP联合提出申请，若申请得到PNP的批准，则可得到BC省提名获批函，之后便可向联邦递交PR申请，获批后申请人及其配偶、子女将会得到枫叶卡获取永久居留身份。联邦移民部对于省提名项目的申请一向可以优先处理。

在BC省雇主担保移民类别下，2017年8月28日BC PNP启动了一项科技行业试点项目Tech Pilot，针对29个紧缺职业申请人，BCPNP目前是每两周发一次邀请，邀请分数比其他职业更低，无需申请LMIA，享有优先处理权。

这29类紧缺职业大部分是ICT和工程师职业，另外还有作家、编辑、广播、音频、电影、表演、平面插图等文化艺术类职业。

Tech Pilot 项目自启动以来一直运作良好，已支持6000多名技术工作者被提名为永久居民。2021年5月28日，BC省宣布BC PNP Tech Pilot项目将无限期延长，并更名为BC PNP Tech。

BC省雇主担保移民评分标准

雇主担保移民虽然相对技术移民来说对申请人的要求略低，但很多项目也是需要申请人进行打分的，满足分数要求才能获得提名省份的邀请，从而进行进一步的申请。

申请BC省雇主担保移民，必须要满足BC省技术移民邀请分，目前BC省是每两周发一次邀请，分别对普通类及Tech类职业申请人进行邀请，目前Tech类职业邀请分在75分左右，普通类邀请分在95分左右。提交注册后，BCPNP会根据申请人提供的信息，从经济因素和人力资本因素两个方面为申请人进行打分，满分为200分。

BC省雇主担保移民雇主要求

做雇主担保移民项目的申请人，一定会很重视雇主的资质，雇主在项目申请过程中起到了非常大的作用，不合格的雇主是无法担保员工申请移民的。飞出国提供的雇主资源都是经过甄选的一手对接的，在源头上把控雇主质量，另外，也是很重要的一点，在申请过程中，加拿大当地的省级政府也会帮您把一道关，去查验雇主是否符合海外招聘的条件。

BC雇主需要满足什么条件

雇主必须满足特定的要求，以支持申请人的BC PNP申请。雇主有责任去证明自己是满足特定要求的，他们有可能会被要求提供财务报表或其他文件作为佐证。

1）雇主是在BC省被确立为有良好声誉的雇主

  • 雇主必须具有以下业务结构之一：
    • 在BC省或省外注册的公司；或
    • 在BC省注册为有限责任合伙企业；或
    • 合格的公营部门或非盈利部门；
  • 雇主必须是“联邦所得税法”所规定的BC省常驻机构；
  • 雇主在BC省至少已经营一年（如果是为其从事入门和半入门级别的工作，或工作是在BC省的东北发展区，则雇主需要在BC省已经营2年）。

2）雇主的企业中有一定数量的长期全职员工

  • 大温地区的雇主必须至少有5名长期工作的全职（或等同于全职）员工；
  • 大温地区以外的雇主必须至少有3名长期工作的全职（或等同于全职）员工；
  • 这些员工必须在雇主的公司工资单上有显示，雇主可能会被要求提供T4已付薪酬摘要以证明这一要求得到满足；
  • 申请人必须与“就业标准法”所定义的雇主建立员工或雇主关系，注：独立承包商不可被视为长期雇员；
  • 在递交申请前的5年中，以及在BC省提名申请过程中，申请人及其配偶或同居伴侣，在向申请人提供工作offer的BC省企业中不得持有超过10%的股权。

3）雇主有良好的工作场所和商业经营历史

雇主必须财务健全，有良好的工作场所和商业经营惯例，包括遵守所有适用的BC省和联邦就业、劳工、移民、卫生和安全法律条例。雇主有可能会被要求提供由专业注册会计师编制的财务报表。

4）雇主能够签署雇主声明

雇主必须签署一份声明，声明该公司符合雇主要求中规定的所有要求，至少在BC省已经营运作一年或两年（如适用），并遵守BC省和联邦就业、移民、卫生和安全法律条例。打印的签名将不被接受，必须为手写。

雇主联系人必须是公司的雇员或所有者，并获得公司的授权和同意，以支持BCPNP申请，包括雇佣外籍工人的任何和所有要求。如果得不到这一同意，可能拒绝申请或撤回申请人的提名。

5）雇主需满足加拿大境内劳动力市场招聘要求

雇主必须证明在当地劳动力市场做出了真诚的招聘努力，雇佣申请人不会对当地候选人（加拿大公民或PR）的就业或发展机会产生不利影响。

经核实后，BCPNP可考虑在下列情况下符合这一要求：

  • 申请人持有有效的工作许可或就业授权，当前正在全职（全年平均每周至少工作30小时）从事BC雇主在工作offer中确定的职业。如申请人当前是持实习工作许可或学习许可在为雇主工作，只有在完成学业之后才可被视为满足该要求；或
  • 雇主可以提供在加拿大境内劳动力市场招聘努力的证据，并能够证明这些招聘工作的结果，例如根据以下最低要求进行的广告：
    • 针对0或A类职业，招聘广告必须至少持续14天。招聘高级管理人员或高度专业化的高级经理或专业人员，可免除14天招聘广告的要求。在这些情况下，雇主必须展示符合行业管理的有针对性的招聘活动，例如设立行政人员招聘小组或雇佣专门的人力资源招聘机构。
    • 针对B类职业，招聘广告必须至少持续14天。
    • 可接受的招聘广告位置包括：被认可的职位公告网站、专业协会网站、全国性报纸、专业杂志或通讯。广告中须包括：公司经营名称、工作职位和职责、薪资范畴、工作地点（局部地区、城市或城镇）、联系信息、工作的技能要求包括：学历、工作经验；
  • 雇主只有努力从当地劳动力市场进行招聘后，才能确定该职位的合格候选人；
  • 真诚的招聘努力包括但不限于：
    • 为确定该职位的加拿大人或永久居民所做的具体努力；
    • 招聘方法和持续时间符合行业标准和做法；
    • 招聘广告中列出的资格条件是合理的，足以使加拿大人或永久居民有资格获得该职位，不要求候选人有海外关系网或海外工作经验，不是只适合于某一特定候选人；
    • 在核实所有潜在候选人的资格时所进行的审查程度；
    • 如果雇主利用第三方的服务进行招聘工作，第三方必须符合省级许可证要求；
  • 请注意，要求员工在BC省以外地区度过大部分时间的职位不符合条件。

6）雇主提供的是一个全职、无固定期限的工作offer

工作offer必须表明申请人已经被录用，并且接受了一个全职（全年平均每周至少30小时）无固定期限的工作。合同工职位是不符合要求的，雇主有责任证明这份工作的真实性。

注：无固定期限工作不适用于BC PNP Tech类别职位

7）雇主能够提供符合行业标准的市场工资率

工作offer中表述的薪资必须以市场率为基准，与BC省具有类似经验和培训水平的工人的工资水平相当，并符合雇主的工资补偿结构。

BCPNP不会将奖金、佣金、分红、小费、加班工资、住房津贴、食宿或其他类似的支付作为工资的一部分。

注：如果一个人的就业影响到劳资纠纷的解决或涉及这类纠纷的人的就业，BCPNP将不会给予提名。

BC雇主需要做什么

除了满足雇主的一般性要求之外，雇主还必须：

  • 以书面形式向申请人提供工作offer；
  • 填妥申请表格的雇主部分；
  • 如果申请人的就业状况有任何变化，如是否已被降职、终止、解雇、长期休假，或该企业是否已关闭或更改所有权，都需要通知BCPNP。

综上所述，BC省雇主担保移民的雇主并不是随随便便选择的，雇主需要提供企业信息及人员架构信息给政府部门，雇主必须是在当地经营良好的企业，不符合条件的雇主无法对外籍工人进行担保。

BC省雇主担保移民申请流程

BC省雇主担保移民可以走EE快读通道，也可以走省提名纸质审理通道，具体走哪个通道取决于申请人的雅思成绩，以及是否满足联邦技术移民67分，也就是说需要首先判断申请人是否满足进入EE池的条件。申请过程中，LMIA不是必需的环节，Tech类职业无需申请LMIA，享有快速审理的特权。

下面我们根据是否需要申请LMIA，将BC省雇主担保移民的流程分类两类。

BC省雇主担保移民申请流程（无需LMIA）

  1. 签约启动；
  2. 获得BC省雇主offer（2个月内）；
  3. 准备材料，递交省提名（1个月）；
  4. 省提名获批，递交工签及PR移民签（3个月）；
  5. 工签获批，登陆加拿大工作（2-3个月）；
  6. 获得PR移民身份（快速类8-12个月；普通类12-16个月）。

BC省雇主担保移民申请流程（需要LMIA）

  1. 签约启动；
  2. 获得BC省雇主offer（2个月内）；
  3. 准备材料，递交LMIA申请（1个月）；
  4. LMIA获批，递交工签及省提名申请（2-3个月）；
  5. 工签及省提名获批，登陆工作，递交PR签证申请（2-3个月）；
  6. 获得PR移民身份（快速类8-12个月；普通类12-16个月）。
